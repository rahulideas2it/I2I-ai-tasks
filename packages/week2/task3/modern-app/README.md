# Modern Node.js Express App

## 📦 Setup

```bash
git clone <repo-url>
cd modern-app
npm install
cp .env.example .env
```

## 🚀 Start Dev Server

```bash
npm run dev
```

## 🧪 Run Tests

```bash
npm test
```

## 🔐 Auth Endpoints

- POST `/api/auth/register` – `{ username, password }`
- POST `/api/auth/login` – `{ username, password }`
